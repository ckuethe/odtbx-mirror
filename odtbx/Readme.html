Orbit Determination Toolbox (ODTBX) Release 8.0 (R2017a) Requirements and Contents


This release of The Orbit Determination Toolbox has been tested with MATLAB R2015a. While parts of it may work with earlier versions of MATLAB, the results are not tested or guaranteed. In conjunction with this MATLAB version, the following operating systems have been tested: Windows 7, MacOS X 10.10, and Ubuntu 14.04. Earlier OS versions may work, but results are not tested or guaranteed.

In order to use the Orbit Determination Toolbox (ODTBX) installer you must first download a Java SE Runtime Environment (JRE) if you don't already have one. You can also choose to download the JDK which includes the Runtime Environment, compilers and development tools. You can download a JRE or JDK from: www.oracle.com. (Other flavors/variations of Java have not been tested with ODTBX and JAT.)

The ODTBX Installer allows the user to install:

  1. Orbit Determination Toolbox Release 8.0
    Once the code is installed everything will be set up and ready to go. A new toolbox named “Orbit Determination Toolbox” will be located inside of the MATLAB help browser. The only requirement for the ODTBX help is that you have MATLAB. For recent MATLAB versions this will be located in the “Supplemental Software” section of the help browser.
    The ODTBX Help includes help documentation and code for the ODTBX. The MATLAB Java Astrodynamic Toolkit (JAT) Adaptor code is also included in this distribution. In order to use the JAT Adapter code you will need to also install the JAT release contained within this installer. The paths will already be set for the MATLAB/JAT interface.

  2. Java Astrodynamic Toolkit
    Release commit d44a587, created on Aug. 26, 2012.
    This JAT release is specific to ODTBX and can be accessed via the Git repository for this release.
    JAT will run on any platform, but requires a Java Virtual Machine (JVM) of version of 1.7 or newer.

  3. Mice - the NAIF MATLAB interface to the SPICE toolkit
    Version N0064, released June 2010, for Matlab R2016b and earlier.
    Version N0066, released April 2017, for Matlab R2017b and later.
    The Navigation and Ancillary Information Facility (NAIF) offers an information system named "SPICE" to assist scientists in planning and interpreting scientific observations from space-borne instruments. SPICE is also widely used in engineering tasks needing access to space geometry. MICE and SPICE can be downloaded from http://naif.jpl.nasa.gov/naif/toolkit.html.

If you have any questions or problems with this installer you can send email to odtbx@lists.nasa.gov.
